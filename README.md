## 📌 About the Project

This project is a **Simple Calculator application** built using Python as part of my **Python Development Internship**.  
The goal of this project was to strengthen my understanding of Python basics, logical conditions, and user input handling through a practical, hands-on task.

---

## 🔹 Project Overview

The calculator performs basic arithmetic operations through a **command-line interface (CLI)**.  
It allows users to enter numbers, select operations, and view results in real time.

This project helped me apply programming concepts in a real-world style task and improve my problem-solving approach.

---

## 🔹 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 💬 User-friendly command-line interaction
- ✅ Input validation for basic error handling

---

## 🔹 Technologies Used

- **Language:** Python 3.x
- **Interface:** Command Line Interface (CLI)

---

## 🚀 How to Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/ishaqhasan870-Dev/simple-Calculator.py.git
   cd simple-Calculator.py
   ```

2. **Run the calculator**
   ```bash
   python "simple Calculator.py"
   ```
